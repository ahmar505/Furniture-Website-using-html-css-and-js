Furniture Website Using HTML, CSS & JavaScript

A modern and responsive furniture website built using HTML, CSS, and JavaScript. This project showcases stylish furniture products with an attractive UI, smooth navigation, and interactive design.

Features
Responsive design for all devices
Modern homepage layout
Product showcase section
Navigation menu
Smooth scrolling and animations
Clean and user-friendly interface
Technologies Used
HTML5
CSS3
JavaScript
Purpose

This project is created for learning frontend web development and improving UI/UX design skills.
